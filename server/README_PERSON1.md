# Người 1 - Authentication Module

## Phần của bạn (có full code):
- ✅ `services/auth.service.js`
- ✅ `controllers/auth.controller.js`
- ✅ `routes/auth.js`

## Chạy server:
```bash
cd server_thu
npm install
npm run dev
```

## Testing:
Test các endpoint:
- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/me
- PUT /api/auth/profile
