# Người 2 - Booking Module

## Phần của bạn (có full code):
- ✅ `services/bookings.service.js`
- ✅ `services/showtimes.service.js`
- ✅ `controllers/bookings.controller.js`
- ✅ `controllers/showtimes.controller.js`
- ✅ `routes/bookings.js`
- ✅ `routes/showtimes.js`

## Chạy server:
```bash
cd server_tu
npm install
npm run dev
```

## Testing:
Test các endpoint:
- POST /api/bookings
- GET /api/bookings
- DELETE /api/bookings/:id
- GET /api/showtimes/:id
